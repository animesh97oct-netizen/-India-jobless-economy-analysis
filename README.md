# India's Jobless Economy: A State-Level Analysis

**Author:** Animesh Banerjee (with co-authors via Rebounce Apple Programme, 2026)

## What This Paper Does

India's GDP grew at 6–8% annually for a decade. Yet formal employment barely moved.  
This paper asks: **why?**

We analyse employment elasticity across 18 major Indian states, build a  
**Quality of Growth (QoG) Index** to measure whether growth actually reaches workers,  
and stress-test which states face the highest structural unemployment risk.
